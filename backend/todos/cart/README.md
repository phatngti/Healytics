# Cart Feature — Todo Pipeline

This folder contains the implementation pipeline for the **Cart Module** (Add to Cart APIs, cart-item lifecycle, and coupon application).

## Ordering

Files are numbered `01-` through `01-`. Execute sequentially.

## Status Convention

Each file starts with a `Status:` line:
- `🔲 TODO` — not started
- `🔄 IN PROGRESS` — currently being worked on
- `✅ COMPLETED` — done, review the **Completed** section for what was implemented
